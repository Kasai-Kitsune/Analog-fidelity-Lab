    📖 Note & Rhythm Dictionary

    Accepted Note Names

    Standard note letters with optional accidentals (# or b) and octave number (default 4):

    • C, C#, Db, D, D#, Eb, E, Fb, E#, F, F#, Gb, G, G#, Ab, A, A#, Bb, B, Cb, B#
    • Octave: -1 to 9 (e.g., C4, Eb5). If omitted, octave 4 is used.
    • Rests: rest or -

    ⚠ Authentic Hardware Mode
    This mode enforces the exact hardware constraints of original Game Boy and Game Boy Color sound systems. It is deliberately restrictive by design.
    • Maximum 4 channels. No exceptions.
    • GB mode: Channel 1 is pulse+sweep only. Channel 2 is pulse only. Channel 3 is wave (limited 4-bit amplitude, 4 coarse volume steps). Channel 4 is noise only — no pitched notes.
    • GBC mode: Identical APU to GB. The only difference is cosmetic and a note about double-speed CPU availability.
    • Waveform options are locked per channel to match hardware capabilities. You do not choose freely.
    • Volume is enforced to hardware-accurate 4-bit resolution (16 steps) and coarse master levels.
    • Frequency range is clamped to the hardware-accurate range. Notes outside it will error.
    • Channel 4 (noise) requires a different note syntax.

    This is not meant to be easy. It is meant to be accurate.
